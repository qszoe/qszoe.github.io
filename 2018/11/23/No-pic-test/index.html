资料参考中公教育2019版申论教材

命题变化

2000-2009 国考申论统一命题

2010-2018 分开命题：省级一套题，市级一套题

涉及范围：政治、经济、社会、文化、生态文明

2010-2018主题特点

  1. 省市主题时有相似
  2. 涉及多个主题
  3. 热门主题冷角度：较新 + 普遍性热点 + 时政小角度问题

出题规律

考试目的

考察从事机关工作应当具备的基本能力，省级市级各有不同，因此考察范围略有不同

  • 省级：注重思维能力（提出本质问题，多角度思考、权衡各方利益，提出中心思想，指导下级工作）
    • 阅读理解能力：提炼材料观点，揭示本质问题
    • 综合分析能力：对材料的内容、观点、问题进行分析合归纳，多角度思考，做出合理判断和评价
    • 提出和解决问题能力：根据自身经验，对给定材料进行分析，发现和界定问题，做出评估或权衡，提出解决问题的方案和措施
    • 文字表达能力：熟练使用指定语种，运用说明、陈述、议论等方式简明扼要、规范标准地表达问题
  • 市级：注重实践、执行能力（理解、概括上级下发材料的内容和意图，执行上级要求的任务）
    • 阅读理解能力：根据给定材料的内容，把握给定材料各部分之间的关系，对所涉及的观点、事实做出恰当解释
    • 贯彻执行能力：准确理解工作目标和组织意图（需要组织什么活动），依法行政，及时有效的完成任务
    • 解决问题能力：对具体问题作出正确分析判断，提出切实可行的措施或办法
    • 文字表达能力：熟练使用指定语种，对事件、观点进行准确合理的说明、陈述或阐释

能力解析

分析申论考察的能力要求，提炼作答要点，规范答题。

阅读理解能力

读懂和理解材料是申论作答的第一步。必须要立足材料，理解所给材料的主题，摸清材料之间的关系。

  1. 全面把握给定材料的内容：
    • 事实
    • 问题
    • 原因
    • 对策
    • 影响
  2. 准确理解材料含义：重点理清给定划线词句的含义
    • 指代含义
    • 引申含义
    • 隐喻含义
  3. 准确提炼材料观点：
    • 提炼事实：材料列举了什么事情
    • 明确目的：材料行文中列举这个或这些事实的目的
    • 揭示问题：这些事实反映了什么问题
    • 表达观点：理清材料列举的事实之间的联系，材料通过对这些事实进行评论分析，表达了什么样的的观点
  4. 揭示材料所反映的本质问题：
    • 把握问题表象下的本质，区分问题现象、完虐抽原因、问题本质三者之间的关系

综合分析能力

考察载体：综合分析题

答题思路：考量材料的主要内容，结合题目要求具体分析，判断材料价值取向，得出合理结论

  1. 根据题目要求，对材料的全部或部分内容、观点、或问题进行分析归纳
    • 题目的具体作答要求
    • 题目的要求的作答内容
  2. 多角度思考材料内容
    • 问题内部的各要素及各要素之间的关系
    • 问题的外部要素
  3. 做出合理推断或评价
    • 运用判断、推理等逻辑方法和知识得出自己的认识
    • 概念、原因、判断评价、改进建议

贯彻执行能力

考察载体：倾向于应用文写作

答题思路：认清题目作答任务、工作目标和组织意图，掌握材料语境下的客观实际情况，明确题目限定的作答身份及对应职权，采取针对性措施完成工作目标，达成工作意图

  1. 理解工作目标和组织意图
    • 理解对象：一般出题材料是上级机关的工作目标和组织意图的资料
    • 明确材料包括的：方针、政策、文件、决策、部署、指示、目标、任务等
  2. 依法行政的原则
    • 对决策的合法性、对策可行性进行分析、鉴别
  3. 根据客观实际情况
    • 明确题目设定的身份立场
    • 明确应用文的受众
    • 注意应用文的写作格式
  4. 及时有效的完成任务
    • 灵活应对特殊事件，考察情景处理能力

提出和解决问题能力

考察载体：提出对策题

答题思路：迅速发现和界定问题，做出正确评估或权衡，根据问题、原因推出对策，不能脱离给定材料

  1. 发现和界定问题
    • 树立问题的特殊性这一概念
    • 把握问题之间的区别和联系
    • 准确界定材料所反映的问题
  2. 做出评估或权衡
    • 认识问题产生的具体原因
    • 涉及问题所涉及的群体
    • 对策和措施要最大限度地满足各种利益需求
    • 注意方案的可行性和有效性
  3. 提出解决方案和措施

文字表达能力

考察载体：文章论述题

答题思路：提炼材料中的事件观点，把握本质和核心，根据材料反映的问题和提出的观点，提炼总论点和分论点，形成正确的写作思路，采用多种论证、修辞手法，美化修饰语言

  1. 熟练运用多种表达方式
    • 申论主要考察陈述、议论、说明
  2. 用语准确规范，简明畅达
    • 使用与机关行文规范相符的特定词汇、语体、语言方式
    • 文风与机关相符
    • 使用机关通用的规范化政治术语
    • 使用规范的书面语体
  3. 表述思想观点
    • 立足材料
    • 个人独特思考

主要题型

  1. 归纳概括题
    • 基础题型
    • 创新题型
  2. 综合分析题
    • 词句理解
    • 启示型分析
    • 评论型分析
  3. 提出对策题
    • 单一型
    • 复合型
  4. 贯彻执行题
    • 宣传演讲
    • 方案总结
    • 观点主张
    • 灵活写作
  5. 文章论述题
    • 命题作文
      • 全命题
      • 半命题
    • 自由命题作文
      • 给出文章写作主题
      • 给出文章写作话题

申论材料

来源和类型

  • 来源：
    • 互联网
      • 国家重点新闻网站（重要来源）：新华网、人民网等
      • 专业网站
    • 报纸杂志
      • 权威性报纸杂志（重要来源）：人民日报、光明日报、环球人物、半月谈等
    • 官方文献
      • 国家重要会议文件：政府工作报告、”十三五规划“纲要等
  • 类型：
    • 新闻报道
    • 理论文章
    • 访谈
    • 网友留言
  • 特点：
    • 理论性日益增强
    • 不过分强调时效性

材料的作用

  1. 体现主题
    • 引用观点，揭示材料主题
  2. 反映问题
    • 所示材料一定会通过列举事件反映索要讨论的问题
  3. 包含原因
    • 直接解释原因：“由于……”等句式直接列举原因
    • 间接展示原因：通过其他人之口（通常为专家等专业人士）来说明原因
  4. 暗含影响
    • 通过议论句，直接或者间接地说明事物、问题产生的影响、危害、作用和意义
    • 影响包含：
      • 影响
      • 危害
      • 作用和意义
  5. 蕴含对策
    • 直接提示：
      • xxx部门采取了一系列措施、制度等
      • xxx专家建议……
    • 案例启示：
      • 正面经验，可以直接概括
      • 反面教训，需要逆向推导、概括
    • 借原因提示：
      • 材料会分析该现象出现的原因，根据原因分条推导、概括

阅读材料

  1. 标注关键词
    • 问题提示词：引出或者表示材料提示的问题所在
      • 困境、弊端、尴尬、瓶颈、软肋、缺失、不良现象、不足之处……
    • 影响提示词：意义、作用、有利于、不利于……
    • 原因提示词：因素、要素、由……引起、由……导致、因为……所以……
    • 对策提示词：建议、要求、措施、务必、希望、应该、将要……
    • 权威词提示：认为、强调、主张、指出、表示……
  2. 核心句
    • 段落首句
    • 段落尾句
    • 中间句子：承上启下的句子
  3. 材料逻辑结构
    1. 各个材料整体逻辑
      • 常规型
        • 提出问题——分析问题——解决问题
        • 现象或问题—>背景、原因、影响、措施
        • 或者三部分打乱顺序，穿插论述
      • 关联型
        • 由一则材料引出另一则材料的内容
      • 争议型
        • 围绕一个具有争议性的论题组织材料，材料中会有两种观点，且为两种观点提供较为充足的理由
    2. 单个材料内部逻辑
      • 总分关系：总领段+分论点
      • 并列关系：内容完全并列，不分主次
      • 因果关系：
        • 原因表述在前，结果表述在后
        • 先说结果（结论），再分析原因
    3. 根据题目或者作答要求指定的逻辑梳理
      • 题目指定逻辑：
        • “要求：围绕xxx的创作宗旨、作品材质及其艺术追求等方面作答”
      • 作答常见逻辑：作答任务中有明显的、约定俗成的逻辑顺序（如公文写作逻辑）

阅卷标准

  1. 要点

    • 采点得分
      • 要点即关键词，问什么答什么，切记立足材料
        • 通常2分一个要点，如10分的题一般为5个要点
      • 明确作答对象，抓住材料和题目的关键词
      • 明确作答身份：作答时所处的立场和角度，题目中没有明确要求的作答身份时，应以一名普通的政府工作人员的身份作答
    • 要点要求：关键词是否出现

  2. 结构

    注意要点之间的逻辑联系。题干中要求“层次分明，分条表述”。

    • 总分结构
      • 适用于各个要点分属于不同层次
      • 范例：”一、……：（1）……（2）…… 二、……（1）……（2）……“
    • 并列结构
      • 适用于要点之间是并列的关系。一般题干中没有“有条理”、“条理清楚”等限制
      • 范例：“一是……二是……三是……”，“首先……其次……最后……”等

  3. 格式

    根据题目要求来判断作答内容是否考虑行文格式：

    • 考虑格式：
      • 题干中指定作答格式：如“提出设问，每个设问标注序号”
      • 草拟一份《xxxxxxx》
      • 只写标题和正文
      • 指定文种（应用文）：倡议书、演讲稿、通知等
        • 格式要求：标题、称谓、发文机关、成文日期等
    • 不考虑格式：
      • 题干中包含：不考虑格式
      • 提纲
      • 内容要点

  4. 分档

    四类文：分档给分。根据文章内容、语言、结构、卷面判断，具体要求不赘述。

    档次 分数
    一类文 28-38分
    二类文 18-28分
    三类文 8-18分
    四类文 8分以下

  5. 语气

    作答语气应与作答身份对应，与情境相符。语气的限制条件会在题目中给出：

    1. 作答者与对象之间的关系：面对群众或者面对上级等
    2. 任务的特殊性：公关礼仪文书要求具有说服力和感染力，党政机关公文则要求用词严谨、语气严肃等

  6. 字数

    一般会规定字数要求，把握好字数10%的标准，即限制200字以内则最佳字数应该在180-220之间。字数过多多少都会扣分

  7. 文面

    1. 字体不可过大或过小，包含在格子当中，占比三分之二左右，不可超过格子之外
    2. 字体应在格子正中最佳
    3. 字迹清晰，不可使用过于含混的连笔字，不可缺笔画、简写等
    4. 纸面洁净，卷面少于三处涂改。错字用括号括起来，在后一个格子写上正确的字即可

题型特点

可以从申论出题的题干使用词句判断该题属于何种类型的题，对症下药。以下简单列举一些题目关键词和作答标准参考。

归纳概括题

  • 题量 ：1-2道
  • 分值 ：10-20分
  • 类型：基础题型，创新题型
  • 特点 ：与综合分析题融合

基础题型

  1. 归纳概括问题
    • 作答标准：明确主要问题，指出指定材料说明的隐患、困境
    • 题干关键词：
      • “请概括…（指定材料、段落）…的主要问题”
  2. 归纳概括做法
    • 作答标准：概括指定材料的主要措施，具体建议，归纳总结经验
    • 题干关键词：
      • “请概括…（指定材料、段落）…采取的主要措施”
  3. 归纳概括原因
    • 作答标准：概括指定材料列举的主要原因、理由
    • 题干关键词：
      • “请概括…（指定材料、段落）…发生的主要原因”
      • “请说明…（指定材料）…的主要理由”
  4. 归纳概括功能
    • 作答标准：简述指定材料说明的政策措施的功能，积极理念或促进作用
    • 题干关键词：
      • “请说明该…{指定材料）…的功能”

创新题型

  1. 分类总结
  • 作答标准：根据题目要求，分条概括指定材料的内容
  • 题干关键词：
    • “列出……的流程和要求”
    • “对……工作进行分类总结”
    • 酱……归纳为若干类型“
  1. 拟制标题
    • 作答标准：根据指定材料的主要内容拟定标题，注意使用材料内多次出现的关键词
    • 关键词：
      • “在下划线处填入、拟定这则资料的标题（小标题）”
  2. 语句填充
    • 作答标准：联系指定材料上下文进行语句填充，使语句前后语义连贯
    • 关键词：
      • “在下划线处各填一句话，使……语义完整“

综合分析题

  • 题量 ：1-2道
  • 分值 ：10-20分
  • 类型：词句理解、阐释，启示型分析，评论型分析

词句理解

  1. 词语理解
    • 作答标准：对给定资料的中的特定词语、短语进行解释，概括其性质、定义
    • 关键词：
      • “根据材料，谈谈对……的理解”
  2. 句子理解
    • 作答标准：对给定资料的中的特定句子进行理解阐释
    • 关键词：
      • “谈谈对这句话的理解”

启示型分析

  • 作答标准：理解材料中出现的问题和现象，得出经验和教训
  • 关键词：
    • “从……中分别获得哪些启示”

评论型分析

  • 作答标准：针对材料提到的社会现象、事件、措施、或观点，进行评价并作出结论
  • 关键词：
    • “根据给定材料，谈谈你的看法”

提出对策题

  • 题量 ：1道
  • 分值 ：20分
  • 特点：一般较少出现
  • 类型：单一型对策题，复合型对策题

单一型对策

  • 作答标准：针对材料内容，根据题目要求概括材料所描述的问题，或对策
  • 关键词：
    • “提出解决……问题的具体措施 / 解决建议”

复合型对策

  • 作答标准：针对材料内容，根据题目要求概括材料所描述的问题和对策
  • 关键词：
    • “根据给定材料，概括存……存在的具体问题，并提出具体建议”
    • “分析……的原因，并提出解决措施

贯彻执行题

  • 题量 ：1~2道 ，必出
  • 分值 ：20分
  • 类型：通用结构，灵活写作

通用结构

  1. 宣传演讲类
  • 作答标准：根据题目要求确定作答身份和写作范围，注意写作格式，如宣传稿、演讲稿、倡议书等，针对材料内容传达观点、倡议、做法
  • 关键词：
    • “根据给定材料，写一篇题为……的讲话稿”
  1. 方案总结类
  • 作答标准：根据题目要求确定作答身份和写作范围，注意写作格式，如通知、建议书、报告等，传达给定材料中的方针政策、指导、安排工作等。该题型主要用于考察处理日常事务的公文写作
  • 关键词：
    • “根据给定材料，写一份汇报”
  1. 观点主张
  • 作答标准：针对材料内容，对某一事件、问题表明立场、观点、态度或主张
  • 关键词：
    • “根据给定材料，为……栏目撰文”

灵活写作

  • 作答标准：根据题目要求确定作答身份和写作范围，注意写作格式，包括讲解稿、简报、导言、编者按、调查问卷等
  • 关键词：
    • “根据给定材料，分条拟一份调查问卷”

文章论述题

  • 题量 ：1道，1000字左右（900-1100字最优），必考
  • 分值 ：35-40分
  • 特点：考察看待问题的深度和广度，题目呈现贼合理化、发散化和个性化
  • 类型：命题作文，自由命题作文

命题作文

  1. 全命题
    • 作答标准：根据给定材料的主题确定写作范围和用例写作，主题应与给定材料相关但不拘泥于给定材料
    • 关键词：
      • “根据给定材料，联系实际，以……为题写一篇文章”
  2. 半命题
  • 作答标准：根据给定材料的主题确定写作范围和用例写作
  • 关键词：
    • ”请以…（给定题目）…为题写一篇文章。用恰当的文字填充…（给定题目）…省略的部分”

自由命题作文

  1. 给定文章主题
    • 作答标准：根据给定材料的主题确定写作范围，注意材料之间的关系
    • 关键词：
      • “请以…（给定题目）…为主题，写一篇文章”
      • “请围绕…（给定题目）…，写一篇文章”
  2. 给定文章话题
    • 作答标准：理解材料中指定句子在材料中的表里含义，根据对这句话的理解或对该资料的思考写作
    • 关键词：
      • 请深入思考给定材料中划线的句子，自拟题目，自选角度，联系实际，写一篇文章”
